All-in-1 (Rekomendasi)

Versi lengkap — semua fitur streaming + fitur tambahan (manga,komik,donghua). Cocok kalau ingin aplikasi utama.

Ukuran file bisa berbeda tergantung build. Jika menginstal di Android: izinkan "Install from unknown sources".

Anime saja

Build ringan yang fokus hanya pada katalog & pemutar anime — cocok untuk perangkat dengan penyimpanan terbatas.

Versi ini mengorbankan beberapa fitur non-anime untuk menghemat ukuran dan RAM.
