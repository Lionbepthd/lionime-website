# 🦁 Lionime

Website streaming anime modern dengan React.js dan Vercel hosting.

## ✨ Fitur
- 🎬 Streaming anime HD
- 📱 Responsive design
- 🔍 Pencarian real-time
- 📅 Jadwal rilis anime
- 🏷️ Filter berdasarkan genre
- ⚡ Fast performance

## 🚀 Tech Stack
- React 18
- React Router DOM
- Vite
- Lucide Icons
- Vercel Hosting
